HTML이란 무엇인가?

하이퍼텍스트 마크업 언어

(HyperText Markup Language, HTML, 문화어: 초본문표식달기언어, 하이퍼본문표식달기언어)

는 웹 페이지를 위한 지배적인 마크업 언어다.

-> creating web pages



-TAG-

글씨 굵게 : strong
밑줄 : u (underline의 줄임말)
heading : h (hn이 커질수록 글씨는 작아짐)
줄바꿈 : br
정해진 간격 줄바꿈: p

css: 첫번째p 다음에 style="margin-top:N px" 하면 N 픽셀만큼 위쪽에 여백만듦

사진 넣기 : img src="사진이름.사진속성" + width="N %"로 크기 조정
리스트 : li -> 리스트는 2대가 같이 다님(table은 3대)
리스트구분(부모) : ul (li<->ul 하나가 있다면 둘다있어야함) (Unordered List 약자)
자동숫자 넘버링 리스트(부모) : ol (Ordered List 약자)
웹페이지 이름 : title
title과 meta를 묶어 head, 나머지는 body안에..
링크 : a (anchor;닻의 약자)
-> a href="주소" + target="_blank"(링크누르면 새 탭) + title="설명"(링크에 마우스가져다대면 나오는 설명)
